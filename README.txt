Go to folder location of the server.py and client.py in the terminal first.
Once there follow the following steps:

1- Sending file from client to server:
	In terminal run the server with python3 Server.py
	In terminal run the client with python3 Client.py
On Client side use the menu to select upload file to server
Let it run and see results in terminal

2- Download file from server to client:
	In terminal run the server with python3 Server.py
	In terminal run the client with python3 Client.py
On the client side use the menu to select download file from server
Let run and see results in the terminal

3- To end program on client side type 3 into the proram menu
	Note*: any number other than 1-3 will prompt invalid number and ask again for valid number
